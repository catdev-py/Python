Guía de utilización:

1. Abre VS Code.
2. Ve al menú 'File' -> 'Open Folder' y selecciona la carpeta 'Reforzamiento_JS_jQuery'.
3. Revisa los archivos de 'Explicaciones' para repasar los conceptos.
4. Abre cada carpeta de 'Ejercicios' y carga el archivo .html en tu navegador.
5. Modifica el código en .js y .css desde VS Code para ver cómo funciona.
